Padding deals with the text or objects inside a box, it is possible to have a universal padding or specify for each side.

hov- new command that is slightly interactive. I was able to change the hover over the menu button. I have been stuck on the concept of adjusting sizes with px. There are mush more efficient methods to resize objects and text. em Relative to the font-size of the element (2em means 2 times the size of the current font). vw Relative to 1% of the width of the viewport vh Relative to 1% of the height of the viewport % Relative to the parent element.
